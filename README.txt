Segue em anexo os códigos fonte juntamente com os 
executáveis. O código fonte em Java recebe como 
entrada um arquivo .txt, onde na primeira linha é 
determinado se o grafo é direcionado ou não 
direcionado, com "D" ou "ND", respectivamente. 
Após isso, nas linhas seguintes, são definidas 
as arestas no formato (v,w), indicando uma aresta 
entre esses dois componentes. Ao final do código 
em Java, é gerado um novo arquivo "matriz.txt", onde 
se encontra novamente na primeira linha a orientação, 
se é direcionado ou não, seguindo da matriz de 
adjacência composta por 0 ou 1. Esse novo arquivo txt
é a entrada para o código em Python que utiliza a 
biblioteca NetworkX, juntamente com a Matplotlib, 
para apresentar o grafo na tela do usuário.

É importante ressaltar que as opções de manipulação 
da matriz, como calcular o grau de um vértice 
qualquer, não serão ofertadas apenas utilizando os 
executáveis, faz-se necessário executar a aplicação 
java direto do código-fonte.
